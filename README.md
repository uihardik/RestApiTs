## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

Using Nestjs it allow to use node js and TypeScript in similar sturcture. it same code like angular has but allow use server side coding.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
